KeyPathBindingNotificationCenter

public class KeyPathBindingNotificationCenter: NotificationCenter

NotificationCenter upon which keyPath modification notifications are posted and observed. This class cannot be instantiated. Instead, access it via NotificationCenter.keyPathBinding.
